CSS: estilizando a Web

Sobre

O CSS - Cascading Style Sheet - é uma linguagem de estilo criada para formatação do conteúdo HTML de sites da Web. Sem o estilo personalizado do site, o seu navegador irá prover um estilo padrão - e bem feito - do conteúdo do seu site.


Sintaxe

Sintaxe é o estudo das relações de concordância, subordinação e ordem nos quais se apresentam os elementos de uma frase. [2] Para formalizar e propagar essas relações, há a necessidade de estabelecer regras, garantindo a reprodutibilidade da linguagem. No caso do CSS, temos as seguintes terminologias:

  • Seletores
  • Blocos de declarações

Seletor

        O seletor é o elemento (tag HTML), .classes e ou #IDs no qual você deseja aplicar o estilo do texto.

Bloco de declaração

        O bloco de declaração é o local no qual você vai fazer todas as declarações para estilizar aquele elemento. 
        Cada declaração é separada por ponto-e-vírgula (;) e é composto por uma propriedade seguido do seu valor.

Como incluir o CSS ao HTML?

Existem 3 formas de incluir o estilo ao seu arquivo HTML:

  1. Referência Externa (recomendado)
  2. Referência Interna
  3. Referência Inline

Layout

Se dessecarmos uma tag HTML, podemos verificar seus vários nomes que envolvem aquela estrutura.

Padding (Espaço Interno)

O padding é o espaço entre o texto e a borda do elemento.

Margin (Espaço Externo)

A margin é o espaço que está contido ao redor da borda do elemento.
