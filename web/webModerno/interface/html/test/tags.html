Observações

        O texto em itálico é geralmente utilizado para indicar um termo técnico, uma frase de outra língua, um pensamento, etc. 
        Enquanto o texto enfático é utilizado durante a verbalização do leitor. 
        Ambos são representados em itálico, alternando-se o contexto de uso.
        O mesmo é válido para as tags "b" e "strong" as quais são representadas em negrito.
